## ‚ú® Features

### Core Functionality

- **Patient Management**
  - Complete patient profiles with contact information
  - Patient registration via secure registration links
  - Teeth chart and dental records tracking
  - Treatment history and medical notes

- **Appointment Scheduling**
  - Interactive calendar view
  - Multiple appointment types (consultation, cleaning, procedure, follow-up, emergency)
  - Appointment status tracking (scheduled, confirmed, in-progress, completed, cancelled)
  - Duration management and scheduling conflicts prevention

- **Treatment Management**
  - Treatment plans creation and tracking
  - Procedure management
  - Quote generation
  - Invoice creation and payment tracking

- **Multi-Role System**
  - **Admin**: Full system access, subdomain management, subscriptions, analytics
  - **Doctor**: Patient care, appointments, treatment plans
  - **Staff**: Appointment scheduling, patient management

- **Administrative Features**
  - Subdomain management for multi-tenant support
  - Subscription management
  - Reports and analytics dashboard
  - User management and role assignment

## üöÄ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/dental_schedule.git
cd dental_schedule
```

### 2. Install Dependencies

```bash
composer install
```

### 3. Environment Configuration

```bash
cp .env.example .env
php artisan key:generate
```

### 4. Database Setup

Update your `.env` file with database credentials:

```env
DB_CONNECTION=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_database_user
DB_PASSWORD=your_database_password
```

### 5. Run Migrations

```bash
php artisan migrate
php artisan db:seed --class=UserSeeder
```

### 6. Set Permissions

```bash
chmod -R 775 storage bootstrap/cache
```

### 7. Start Development Server

```bash
php artisan serve
```

Visit `http://localhost:8000` in your browser.

## üìñ Usage

### Patient Registration

Patients can register using a secure registration link generated by admin/staff. The link format is:
```
https://yourdomain.com/register/{token}
```

### Creating Appointments

1. Navigate to **Appointments** ‚Üí **Create New**
2. Select patient, doctor, date, and time
3. Choose appointment type and duration
4. Add notes if needed
5. Save appointment

### Managing Treatments

1. Access patient profile
2. Navigate to **Treatment Plans**
3. Create treatment plan with procedures
4. Generate quotes for patient approval
5. Convert to invoices upon approval
6. Track payments

### Calendar View

- View all appointments in calendar format
- Filter by doctor, date range, or status
- Quick access to appointment details
